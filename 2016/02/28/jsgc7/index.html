JavaScript引用类型1--对象、数组

引用类型是一种数据结构，用于将数据和功能组织在一起。
引用类型的值是引用类型的一个实例。
ECMAScript提供了很多原生引用类型，以便开发人员实现常见的计算任务。

Object类型

在应用程序中存储和传输数据的理想选择。
有两种创建Object实例的方法。
第一种使用new操作符后跟构造函数。

var person = new Object();
person.name = "tanChenHao";
person.age = 23;

另一种是使用字面量表示法。

var person = {
    name : "tanChenHao",
    age : 23;
}

属性名可以使用字符串。
如果大括号内留空，则定义的对象实例只包含默认的属性和方法。
对象字面量法是定义对象常用的方法，也是向函数传递大量可选参数的首选方法。
一般访问对象属性时使用点表示法，还可以使用方括号表示法。在使用方括号语法时，应该将要访问的属性以字符串的形式放在方括号中。方括号语法的主要用处是可以通过变量来访问属性，将属性名保存在变量中。

Array类型

ECMAScript数组的每一项可以保存任何数据类型。
数组的大小可以动态调整。
创建数组的方法有两种。第一种是使用Array构造函数。

var colors = new Array();//new操作符可以省略

可以给构造函数传递数值即数组项的数量；可以直接向构造函数传递数组中包含的项。
第二种基本方式是使用数组字面量表示法。

var colors = ["red", "blue", "green"];

数组的length属性不是只读的，可以设置这个属性，从数组的末尾除去项或添加项。

检测数组

ECMAScript新增了Array.isArray()方法，可以确定某个值到底是不是数组，而不管它在哪个全局环境中创建的。

if(Array.isArray(value)){
    //对数组执行某些操作
}

转换方法

所有对象都具有tolocaleString()、toString()和valueOf方法。
数组的toString()和valueOf()会返回数组中每个值的字符串形式拼接而成的字符串，以逗号分隔。不同之处在于valueOf()会调用数组每一项的toString.
toLocaleString()方法也经常返回和toString()相同的值，不同在于每一项调用的是tolocaleString()方法。
join()方法，使用不同分隔符来创建这个字符串，接收一个参数即用作分隔符的字符串。

栈方法

栈是一种LIFO的数据结构，即最新添加的项最早被移除。ECMAScript为数组提供了push()方法和pop()方法，以便实现栈的行为。
push()方法可以接收任意数量的参数，把他们逐个添加到数组末尾，并返回修改后数组的长度。
pop()方法则从数组末尾移除最后一项，减少数组的length值，然后返回移除的项。

队列方法

队列是一种FIFO的数组结构。队列在列表的末端添加项，从列表的前端移除项。使用push()和shift()f方法就能和队列一样使用数组。
shift()方法能移除数组中第一个项并返回该项，同时数组长度减1.
unshift()方法能在数组前端添加任意个项并返回新数组的长度。
使用unshift()和pop()可以实现前端添加，末端移除的功能。

重排序方法

数组中有两个直接用来重排序的方法：reverse()和sort().
reverse()方法可以反转数组项的顺序。
sort()方法可以按升序排列数组，但是sort()会调用数组的toString()方法，通过字符串排序，会导致排序错误。
sort()方法可以接受一个比较函数作为参数，以便我们进行正确的排序。

function compare(value1, value2){
    //可以通过更改比较的结果，得到降序排列的结果
    if (value1 < value2){
        return -1;
    }else if(value1 > value2){
        return 1;
    }else{
        return 0;
    }
}

var values = [0, 1, 5, 10, 15];
values.sort(compare);
alert(values);

对于数值类型或者其valueOf()方法会返回数值类型的对象类型，可以使用简单的比较函数。

function compare(value1, value2){
    return value2 - value1;
}

操作方法

concat()方法可以基于当前数组所有项创建一个新数组。具体来说，这个方法会先创建当前数组的一个副本，然后把接收到的参数添加到这个副本的末尾，最后返回新构建的数组。

slice()方法可以基于当前数组中的一个或多个项创建新数组。它可以接收一或两个参数，即要返回项的起始位置和结束位置，返回起始位置和结束位置之间的项（不包括结束位置的项）。如果只有一个参数即起始位置，就返回起始位置到数组末尾所有的项。

splice()方法，可以用于向数组中删除、插入、替换项。splice()始终会返回一个数组，该数组包含从原数组中删除的项。

  • 删除：两个参数：起始位置，要删除项的数量。
  • 插入：三个参数：起始位置，0，要插入的项。
  • 替换：三个参数：起始位置，要删除项的数量，要插入的项。

位置方法

ECMAScript5为数组实例添加了两个位置方法：indexOf和lastIndexOf().这两个方法都有两个参数：要查找的项，查找开始的位置（可选）。不同之处，indexOf()从数组的开头向后找，lastIndexOf()从数组的末尾开始向前找。比较时都使用全等操作符，返回要查找的项在数组中的位置，没有找到返回-1.

迭代方法

ECMAScript5为数组定义了5个迭代方法。每个方法都接收两个参数：要在每一项上运行的函数，运行该函数的作用域对象（可选）。传入这些方法中的函数会接收三个参数：数组项的值（item）,该项在数组中的位置（index）,数组对象本身（array）。

  • every():对数组中的每一项运行给定函数，如果该函数对每一项都返回true，则返回true。
  • filter():对数组中每一项运行给定函数，返回该函数会返回true的项组成的数组。
  • forEach():对数组中的每一项运行给定函数。没有返回值。本质上与for循环迭代数组一样。
  • map():对数组中的每一项运行给定函数，返回每次函数调用结果组成的数组。
  • some():对数组中的每一项运行给定函数，如果该函数对任一项返回true，则返回true。

归并方法

ECMAScript5还新增了两个归并数组的方法：reduce()和reduceRight().这个两个方法度会归并数组所有项，然后构建一个最终返回的值。其中，reduce()方法从数组第一项开始，而reduceRight()从最后一项开始。
这两个方法都接收两个参数：每项上调用的函数，归并初始值（可选）。
传入非函数接收4个参数：前一个值（prev），当前值（cur）,项的索引（index），数组对象（array）。

var values = [1, 2, 3, 4, 5];
var sum = values.reduce(function(prev, cur, index, array){
    return pre + cur;
    })
alert(sum); //15
