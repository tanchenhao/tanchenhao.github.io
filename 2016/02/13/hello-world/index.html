mac下hexo建立github博客

安装git

安装node.js

我的git版本git version 2.5.4 (Apple Git-61)
我的是node-v5.6.0

在github建立新的仓库

名字为：yourname.github.io

安装hexo

npm install -g hexo 
sudo su //进入root用户模式

退出root用户模式
hexo init blog //在blog文件夹下新建配置文件
npm install

连接github

修改blog文件夹下_config.yml文件(注意冒号后要空格)
deploy:
type: git
repository: https://github.com/yourname/yourname.github.io.git
branch: master

npm install hexo-deployer-git --save//安装连接插件
hexo generate //创建hexo
hexo deploy  //上传hexo

在浏览器输入github.yourname.io即可浏览

本地浏览

cd blog//进入博客所在文件夹
hexo server//开启本地服务器

提示:Hexo is running at http://0.0.0.0:4000/. Press Ctrl+C to stop.
在浏览器输入http://0.0.0.0:4000/即可本地浏览
