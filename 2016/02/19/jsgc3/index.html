JavaScript基本概念1

标识符

标识符（指变量、函数、属性的名字，函数的参数）

  • 第一个字符必须是字母，下划线或者美元符号；
  • 其他字符随意。
    ECMAScript标识符采用驼峰命名，第一个字母小写，其余字母首字母大写。

严格模式

在严格模式下，ECMAScript3 中的一些不确定的行为将得到处理，而且对某些不安全的操作也会抛出错误。
要在脚本中启用严格模式，可以在顶部添加如下代码：
“use strict”;

变量

用var操作符定义的变量将成为定义该变量的作用域中的局部变量。虽然省略var操作符可以定义全局变量，但不推荐这么做，会造成不必要的混乱使代码难以维护。
可以使用var定义多个变量，用逗号分开即可
var message = “hi”,
found = false,
age = 29;

数据类型

ECMAScript中有5种数据类型（也称基本数据类型）：

  • Undefined
  • Null
  • Boolean
  • Number
  • String
    还有一种复杂的数据类型————Object，Object本质上是由一组无序的名值对组成的。

typeof操作符

用来检测给定的变量的数据类型，typeof操作符的操作数可以是变量，也可以是数值字面量。对一个值使用typeof操作符可能返回下列某个字符串：

  • “undefined”————未定义
  • “boolean”————布尔值
  • “string”————字符串
  • “number”————数值
  • “object”————对象或null
  • “function”————函数
    调用typeof null会返回”object”,因为特殊值null被认为是一个空的对象引用。

Undefined类型

使用var声明变量但未对其加以初始化时，这个变量的值就是undefined。对未初始化和为声明的变量执行typeof操作符都返回undefined值，但只要我们能做到定义变量时完成初始化，我们就就知道被检测的变量还没被声明。

Null类型

null值表示一个空对象指针，而这也是typeof null返回”object”的原因。如果定义的变量准备在将来用于保存对象，那么最好将该变量初始化为null而不是其他值。实际上undefined值是派生自null，所以相等性测试时会返回true。

Boolean类型

该类型有两个字面值：true和false，这两个值和数字值不是一回事，true不一定等于1，而false也不一定等于0。调用Boolean（），可以返回这个值对应的Boolean值。至于返回的值是true还是false，取决于要转换的数据类型及其实际值。流程控制语句（如if语句）自动执行相对应的Boolean转换。

Number类型

采用IEEE754格式来表示整数和浮点数。最基本的数值字面量格式是十进制整数，正式还可以通过八进制数值前加0，十六进制数值前加0x，如果超过了范围，将被当做十进制解析。在进行算数计算时，所有以八进制或者十六进制表示的数值都会被转换成十进制数值。

浮点数值

浮点数值就是该数值中必须包含一个小数点，并且小数点后必须有一位数字，否则ECMAScript会把浮点数值转换为整数值，为了节省一半内存空间。对于极大的或者极小的数用e表示法。浮点数的最高精度是17位小数，但进行计算时其精度远远不如整数。如0.1+0.2的结果不是0.3。所以永远不测试某个特定的浮点数值。

数值范围

ECMAScript的最小数值是5e-324保存在Number.MIN_VALUE中，最大数值是1.7976931348623157e+308保存在Number.MAX_VALUE。如果计算得到一个超过JavaScript数值范围的值，负数转换为-Infinity,正数转换为Infinity，而且无法参与下一次计算。
使用isFinite（）函数来判断是否处于数值范围内，处于最大最小值之间则返回true。

NaN

NaN即非数字是一个特殊的数值，表示一个本来要返回数值的操作数未返回数值的情况。0除以0返回NaN，任何涉及NaN的操作返回NaN.NaN与任何值不相等包括NaN本身。
isNaN()函数，当接收的参数不是数值时返回true。isNaN()在接收到一个参数时会自动进行数值转换，再进行判断。

数值转换

有3个函数可以把非数值转换为数值：Number（）、parseInt（）、parseFloat（）。
Number（）的转换规则如下：

  1. Boolean值，true转换为1，false转换为0.
  2. 数字值，传入什么返回什么。
  3. null，返回0.
  4. undefined, 返回NaN。
  5. 字符串，字符串中只含有数值，无论八进制十六进制都将转换为十进制数值；浮点格式转换为对应的浮点数值；字符串为空转换为0；字符串中有除了数值的其他字符转换为NaN.
  6. 如果是对象，则调用对象的valueOf()方法，然后按照上面字符串的转换规则返回相应的值。如果返回了NaN，则调用toString()方法，然后再按照前面的转换规则转换返回的字符串值。
    parseInt()的字符串转换规则如下：
  7. 如果第一个字符不是数字字符或负号，返回NaN.空字符串会返回NaN.

  8. 如果第一个字符是数字字符，继续解析下一个字符，直到解析完成或者遇到非数字字符.（小数点是非数字字符）
    ECMAScript3中能识别八进制十六进制数。ECMAScript则不能。
    parseInt()提供第二个参数：使用多少进制来解析。为了避免错误，设置第二个参数是非常必要的。
    parseFloat()字符串转换规则：
    与parseInt()类似，只是parseFloat()能够识别小数点。还有，parseFloat()只接受一个参数，只能进行十进制解析。

    String类型

    即字符串可以用””或’’表示，表示的字符串完全相同。ECMAScript字符串是不可变的，如果要改变字符串，首先要销毁原来的字符串，然后用另一个包含新值得字符串填充该变量。

    字符字面量

    列如：\n, \t, \b 将被当做一个字符来解析。
    任何字符串的长度凑可以其length属性取得。

    转换为字符串

    要把一个值转换为字符串有两种方式。第一种是使用几乎每个值都有的toString()方法。
    数值、布尔值、对象和字符串值都有toString方法。但null和undefined没有。在调用数值的toString方法时可以传递一个参数：转换为多少进制。
    如果不能确定转换的值不是null或undefined，可以使用转型函数String(),参数为要转换的值。这个函数先调用toString()方法，如果是null或undefined则返回对应的null货undefined。

    Object类型

    对象可以通过执行new操作符后更要创建的对象名称来创建。创建Object类型的实例并为其添加属性和方法，就可以创建自定义对象。Object类型是所有他的实例的基础，Object类型所具有的任何属性和方法同样存在于更具体的对象中。
    Object的每个实例都具有下列属性和方法.

    • constructor:保存着用于创建当前对象的函数。
    • hasOwnProperty(propertyName):用于检查给定的属性在当前对象实例中（而不是在实例的原型中）是否存在。其中，作为参数的属性名（propertyName）必须以字符串形式指定。
    • isPrototypeOf(object):用于检查传入的对象是否是传入对象的原型。
    • propertyIsEnumberable(propertyName):用于检查给定的属性是否能够使用for-in语句来枚举。作为参数的属性名必须以字符串形式指定。
    • toLocaleString():返回对象的字符串表示,该字符串与执行环境的地区对应。
    • toString():返回对象的字符串表示。
    • valueOf():返回对象的字符串、数值、或布尔值表示。通常与toString()方法的返回值相同。
