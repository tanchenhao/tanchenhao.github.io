html学习笔记

Html把网页上的内容分成大大小小的盒子。

  • html标记
  • 块级元素和行内元素
  • 文档对象模型

a标签

target规定在何处打开链接

  • View_window 标记的链接都在同一个新窗口中打开
  • View_frame 在框架中打开窗口
  • _blank 总是以新窗口打开链接
  • _self 默认值，在本窗口打开链接

    form 标签

    action属性规定向何处发送表单，值为url。
    method规定表单处理数据的方式，值post/get。
    name规定表单的名字。
    target规定在何处打开表单action中的url。

input标签

type属性规定input类型

  • button可点击按钮
  • checkbox定义复选框
  • radio定义单选按钮
  • text定义文本框
  • password定义密码框
  • file定义输入字段和浏览按钮，上传文件
  • hidden定义隐藏的输入字段
  • submit定义提交按钮
  • reset定义重置按钮
  • image定义图像形式的提交按钮+src

    name属性规定 input 元素的名称

    用于对提交到服务器后的表单数据进行标识，或者在客户端通过 JavaScript 引用表单数据。
    注释：只有设置了 name 属性的表单元素才能在提交表单时传递它们的值。

    value规定input的值

    type=”button”, “reset”, “submit” - 定义按钮上的显示的文本
    type=”text”, “password”, “hidden” - 定义输入字段的初始值
    type=”checkbox”, “radio”, “image” - 定义与输入相关联的值

textarea 标签定义多行的文本输入控件

table标签定义表格

标签定义 HTML 表格中的行

HTML 表单中有两种类型的单元格：

  1. 表头单元格 - 包含表头信息（由 th 元素创建）
  2. 标准单元格 - 包含数据（由 td 元素创建）
