mac下hexo建立github博客

##1. 安装git
我的git版本git version 2.5.4 (Apple Git-61)

##2. 安装node.js
我的是node-v5.6.0

##3. 在github建立新的仓库
名字为：yourname.github.io

##4. 安装hexo
npm install -g hexo 进入root用户模式 sudo su
退出root用户模式
hexo init blog（自定义文件夹名）
npm install

##5. 连接github
修改blog文件夹下_config.yml文件(注意冒号后要空格)
deploy:
type: git
repository: https://github.com/yourname/yourname.github.io.git
branch: master
安装连接插件
npm install hexo-deployer-git –save
修改 hexo generate
上传 hexo deploy
在浏览器输入github.yourname.io即可浏览

##6. 本地浏览
进入博客所在文件夹 cd blog
开启本地服务器hexo server
提示Hexo is running at http://0.0.0.0:4000/. Press Ctrl+C to stop.
在浏览器输入http://0.0.0.0:4000/即可本地浏览
